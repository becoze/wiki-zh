### 数组遍历 和 循环

```java
double[] arr = {1.0, 2.0, 3.0, 4.0}; // 静态初始化

for(int i = 0; i < arr.length; i++){
	System.out.println(name[i]);
}
```

### 下标越界异常 Array_Index_Out_of_Bounds_Exception

- 下标 index：表示第`i`个元素的值。下标是从0开始编号的，第一个元素就是`name[0]`。
- 编译不会报错，运行会报错。
- 数组的下标是从0开始的
    - `int[]arr=new int[5];`则有效下标为0-4
