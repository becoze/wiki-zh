# 属性 Field

## 属性

<aside>
💡 **属性**，也叫**成员变量**，或**字段** field。*Properties*

</aside>

### 属性的类型

```java
class Cat{
	String name;    // 引用类型
	String[] master;
	double price;   // 基本类型
	int code;
}
```

[⭐基本数据类型 Data Type](https://www.notion.so/Data-Type-7decffbb79da4fde9b84dcf864ffcc0a?pvs=21) 

属性的类型可以是：

- 基本数据类型（如：int，double，boolean）
- 引用数据类型（如：类/对象 - String，数组）

### 属性有默认值

属性如果不赋值，有默认值，规则和基本数据类型/数值一致。
